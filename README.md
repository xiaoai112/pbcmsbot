# AIGOU 智能文章生成与自动发布后台

AIGOU 是一套面向内容运营、站群管理和 PbootCMS 网站自动发布场景的智能文章生成后台系统。系统支持多用户注册、会员管理、大模型文章生成、违禁词替换、图片素材管理、定时发布、PbootCMS 站点桥接发布、锚文本随机插入、站点发布统计、邮件通知以及易支付会员充值等功能。

本项目适合用于企业官网内容更新、SEO 软文生产、站群内容管理、自动化发布后台、会员制内容发布工具等场景。

## 主要功能

### 1. 多用户系统

- 支持用户注册和登录
- 支持管理员账号和普通会员账号
- 管理员可管理所有会员
- 会员可独立管理自己的站点、文章、图片、任务和模型配置
- 支持会员启用、禁用、删除
- 支持管理员重置会员密码
- 支持管理员和会员自行修改登录密码
- 支持会员填写邮箱，用于接收文章发布成功通知

### 2. 会员充值与易支付收款

- 集成易支付收款功能
- 只有管理员可以配置易支付参数
- 支持配置：
  - 易支付接口地址
  - 商户 ID
  - 商户 Key
  - 收款站点名称
- 支持自定义会员套餐
  - 周卡
  - 月卡
  - 季卡
  - 年卡
  - 自定义天数
- 会员端支持在线充值
- 支持支付宝、微信支付、QQ 钱包等易支付兼容支付方式
- 支付成功后自动延长会员有效期
- 支持支付订单记录
- 支持易支付异步回调
- 回调防重复处理，避免重复增加会员时长

### 3. 大模型文章生成

- 用户可自行配置大模型 API 地址和 API Key
- 支持 OpenAI 兼容接口
- 支持读取接口支持的模型列表
- 支持自由切换模型
- 支持多个关键词批量生成文章
- 文章按关键词逐篇生成，不一次性强行生成
- 可指定目标生成数量
- 生成后的文章进入“我的文章”统一管理

### 4. 违禁词管理

- 支持后台维护违禁词替换规则
- 可为不同用户保存不同违禁词库
- 支持常用违禁词模板
- 文章生成和发布前可替换高风险词
- 适合规避广告法相关高风险表述，例如“最、第一、唯一、国家级”等

### 5. 我的文章管理

- 查看已生成文章
- 查看文章状态
- 手动发布文章
- 删除单篇文章
- 批量删除文章
- 已发布文章会标记为“已发送”
- 每篇文章默认只发送一次
- 支持文章内容完整预览

### 6. 图片管理

- 支持上传文章配图
- 支持图片缩略图展示
- 支持批量上传
- 支持批量删除
- 发布文章时可使用图片素材
- 支持通过后台接口访问图片资源

### 7. 站点管理

- 支持添加多个目标站点
- 支持 PbootCMS 站点桥接发布
- 支持配置：
  - 站点名称
  - 站点域名
  - CMS 类型
  - PbootCMS 发布接口地址
  - 接口 Token
  - 发布栏目
- 支持读取 PbootCMS 栏目信息
- 支持站点连接测试
- 支持站点发布统计

### 8. PbootCMS 桥接发布

- 提供 PbootCMS 桥接文件示例
- 支持通过桥接接口发布文章
- 支持读取目标站栏目
- 支持向指定栏目发布文章
- 支持图片随文章发布
- 支持发布成功后记录状态

### 9. 锚文本管理

- 支持为站点配置锚文本
- 支持自定义关键词和链接
- 发布文章时可随机插入一个锚文本
- 支持启用或停用锚文本
- 有利于 SEO 内链建设

### 10. 定时发送

- 定时发送不再重新生成文章
- 从“我的文章”中抽取指定数量的待发送文章
- 每篇文章只发送一次
- 发送成功后标记为“已发送”
- 支持每天固定时间执行
- 支持指定发送数量
- 支持手动执行定时任务
- 支持暂停、启用、删除任务

### 11. 发布日志

- 记录站点接入、文章生成、文章发布、充值、会员管理等操作
- 支持查看成功和失败状态
- 方便排查发布异常

### 12. 邮件通知

- 只有管理员可以配置 SMTP 邮件通知
- 会员只需要填写自己的邮箱
- 文章发布成功后，系统自动向对应会员邮箱发送通知
- 支持发送测试邮件
- 支持 SMTP 地址、端口、SSL、用户名、授权码配置

### 13. 系统设置

- 管理员可配置网站名称
- 支持更换 Logo
- 支持修改首页标题和介绍文案
- 支持首页广告位管理
- 支持广告标题、文案、图片、跳转链接、排序和启停
- 登录页和后台底部带版权说明

## 技术栈

- 前端：React + Vite
- 后端：Node.js 原生 HTTP 服务
- UI 图标：lucide-react
- 邮件：nodemailer
- 数据存储：
  - 默认 JSON 文件存储
  - 支持 MySQL 存储模式
- 进程管理：PM2
- Web 服务：Nginx 反向代理
